# The Few

> "Never in the field of human conflict was so much owed by so many to so few." — Churchill

A personal advisory board. Follow a small group of thinkers, track their latest output, and consult their perspectives when you make decisions.

## Components

- **Pulse** — track what they're saying now
- **Council** — ask what they'd think

## Setup

```bash
pip install -e .
cp config/people.example.toml config/people.toml
python -m the_few.pulse  # first run: fetch and summarize
```
